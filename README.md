# Portfolio – Yeray Sánchez

Este repositorio contiene mi portfolio personal como estudiante del Grado Superior en Desarrollo de Aplicaciones Web (DAW). Incluye información sobre mí, mis habilidades, mis proyectos y mis datos de contacto. Su objetivo es servir como presentación profesional para empresas y mostrar mi progreso como desarrollador web.

---

## Descripción del proyecto

El portfolio está desarrollado utilizando HTML y CSS, con un enfoque en diseño responsive y una estructura clara. El propósito es presentar de forma profesional mi perfil como desarrollador en formación.  
El sitio está desplegado mediante GitHub Pages para permitir el acceso desde cualquier dispositivo sin instalación previa.
